# 自動身分組

設定一個自動化的身分組系統，成員只需要點選訊息下的按鈕就可以獲得身分組！

![自動身分組](/img/docs/role/role.png)

## 初始化

<Adsense/>

首先使用 `&role` 指令彈出詳細說明，點選 `開始設定` 來初始化，別擔心你可以設定很多個訊息！

![初始化 自動身分組](/img/docs/role/setup.png)

之後會彈出一個下拉選單可以選擇你想自動化的身分組，你想要幾個就可以選擇幾個。

:::info 我想先提醒你
因為 Discord 官方的限制，下拉選單以及按鈕最多是 25 個為限制，這點還請見諒。
:::

![下拉選單 自動身分組](/img/docs/role/menu.png)
