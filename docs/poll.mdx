線上調查可以了解大眾對一件事情或主題的看法，當你在運行一個 Discord 群組並且進行像是群組內的大型投票、選舉等競爭的時候，可以使用投票功能，來去快速的決議出結果。

## 如何發起投票

YEE式機器龍提供了方便美觀且可以有許多自定義選項的投票功能，讓你可以快速決議群大小事！

首先在要發起投票的頻道使用 `&poll` 指令，將會跳出創建投票的方法。

<Adsense/>

![投票 指令](/img/docs/poll/poll-command.png)

我們使用一個簡單的範例來創建投票：

```
&poll 可口可樂vs百事? 可口可樂 百事
```

上面這段指令的意思是：創建一個投票 標題為 `可口可樂vs百事?`，並且有兩種選項 `可口可樂` 跟 `百事`。

那如果在標題或是選項欄位有需要空格可以使用 `"` 引號將整個欄位括弧起來：

![投票 範例](/img/docs/poll/poll-example.png)

<Adsense/>

## 投票者操作選項

在投票的下拉選單中，發起人可以編輯該投票的自定義設定。

![投票 操作選項](/img/docs/poll/poll-operation.png)

### 允許變更選項

> 預設：`關閉`

切換這個選項可以切換一般使用者是否可以改投其他的選項，假設說有成員投錯了可以開啟這個選項讓他可以更換。

### 公開投票結果

> 預設：`不公開`

要查詢投票結果可以按投票最後一顆 `查看投票結果` 的按紐來查詢，點擊查詢結果之後將會顯示出投該選項的使用者標記，該選項投票佔的比數，以及下方的圓餅圖。

若沒有切換成公開時，除了發起人沒有人可以看到投票結果。

<Adsense/>

![投票 結果](/img/docs/poll/poll-result.png)

:::info 我想先通知你
因為訊息長度限制，一個選項最多顯示出前 `40` 個投票者。
:::

### 更新投票

重新渲染投票結果

若是你遇到了一些BUG，你可以使用這個選項來更新投票訊息

### 移除指定使用者投票

移除特定某個成員投的票

你可以透過 `&userinfo` 指令來獲得使用者的 ID

![投票 移除](/img/docs/poll/poll-remove.png)

### 結束投票

發起投票者可以點選結束投票來中止這個投票。

:::warning 我想先警告你
投票結束之後將沒辦法再做復原！不過還是可以切換是否公開以及查看投票結果。
:::
