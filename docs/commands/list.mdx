透過使用指令，你能夠跟機器人進行互動，並且同時又不干擾到聊天，使用 `&` 開頭呼叫這些指令，或是標記 `@YEE式機器龍` 也可以

## 一般

| 指令                  | 描述                    |
|---------------------|-----------------------|
| &user [使用者標記]       | 取得使用者資訊               |
| &guild [伺服器ID]      | 取得伺服器資訊               |
| &mc [玩家ID / 伺服器IP]  | 查詢Minecraft玩家 / 伺服器資訊 |
| &ping               | 確認我是不是活著              |
| &help [指令]          | 查詢指令的用法 / 教學          |
| &yt <頻道名稱 / ID>     | 查詢YouTube頻道資訊         |
| &dvc                | 動態語音頻道                |
| &bullshit <主題> [字數] | 唬爛產生器                 |
| &clear <數量>         | 大量刪除訊息                |
| &pick <主題>          | 幫你選一個答案               |
