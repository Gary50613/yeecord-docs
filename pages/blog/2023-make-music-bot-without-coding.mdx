## 創建 Discord 機器人

如果你已經有了一個 Discord 機器人，你可以跳過這個區段
<div className='steps-container'>

    ### 創建 Application

    前往 [Discord Developer Portal](https://discord.com/developers) 並登入你的帳號，接著點擊 New Application

    ![developer-portal.png](@static/blog/2023-make-music-bot-without-coding/developer-portal.png)

    ### 為機器人取名

    這個名字將會顯示在 Discord 上，選一個你覺得好聽的名字吧！

    ![name-your-application.png](@static/blog/2023-make-music-bot-without-coding/name-your-application.png)

    ### 獲得 Token

    到左方的 Bots，點擊 Reset Token 獲得 Token

    Token 會看起來像這樣：<br/>
    `MTA5ODYxNDI5NjExMzU3Nzk5NA.Ga7fZq.j9XXSVOWDdXJIlbZpOrkL7fsT4xB42zlHjB7Fg`<br/>
    請將這個字串儲存起來，我們稍後會使用到

    ![get-token.png](@static/blog/2023-make-music-bot-without-coding/get-token.png)

    <Warning title="警告">
        請不要將這串 Token 分享給任何人，這就像是你機器人的帳號密碼，得到了它，就可以肆意控制你的機器人
    </Warning>

    ### 開啟 Intents

    從剛剛的頁面往下捲，開啟 Intents<br/>
    機器人需要這些 Intent 才能夠運作，將三個 Intent 開啟後，請記得點擊 Save Changes

    ![enable-intents.png](@static/blog/2023-make-music-bot-without-coding/enable-intents.png)

    ### 獲得機器人邀請連結

    請勾選 `bot` 和 `applications.commands`

    ![setup-scopes.png](@static/blog/2023-make-music-bot-without-coding/setup-scopes.png)

    並捲到下方，點擊 `Copy` 來複製邀請連結

    ![copy-invite-link.png](@static/blog/2023-make-music-bot-without-coding/copy-invite-link.png)
</div>

## 設置 Spotify

這是為了對 [Spotify](https://open.spotify.com/) 連結的支援，以及自動播放，
如果你不需要這些功能，你可以跳過這個區段

<div className='steps-container'>

    ### 創建 Spotify 應用程式

    前往 [Spotify Developer Dashboard](https://developer.spotify.com/dashboard/)，並登入你的 Spotify 帳號，接著 Create
    App

    ![create-spotify-app.png](@static/blog/2023-make-music-bot-without-coding/create-spotify-app.png)

    ### 創建 App

    `App name`, `App description`, `Website` 可以隨意填寫，但 `Redirect URIs` 請填入 `http://localhost/`

    填完後點擊 `Save`

    ![fill-in-forms.png](@static/blog/2023-make-music-bot-without-coding/fill-in-forms.png)

    ### 獲得 Client ID 和 Client Secrets

    進入 Settings

    ![get-into-settings.png](@static/blog/2023-make-music-bot-without-coding/get-into-settings.png)

    點擊 View Client Secrets

    ![show-client-secrets.png](@static/blog/2023-make-music-bot-without-coding/view-client-secrets.png)

    複製 Client ID 和 Client Secrets 並儲存，我們稍後會使用到

    ![copy-fields.png](@static/blog/2023-make-music-bot-without-coding/copy-fields.png)

    <Warning title="警告">
        Client ID 和 Client Secrets 也是私密資訊，任何人只要得到了它，就可以肆意控制你的 Spotify 帳號
    </Warning>
</div>

## 開始架設 Lava

<div className='steps-container'>

    ### 下載 LavaLauncher

    你可以使用 `git clone` 或是直接前往 [LavaLauncher](https://github.com/Nat1anWasTaken/LavaLauncher) 下載 ZIP 檔

    ```bash
    git clone https://github.com/Nat1anWasTaken/LavaLauncher.git
    ```

    ![下載 LavaLauncher](@static/blog/2023-make-music-bot-without-coding/get-lavalauncher.png)

    記得將下載下來的 ZIP 檔案解壓縮到一個你喜歡的地方

    ### 運行腳本
    <Tabs items={["Windows", "Linux / Mac", "Pterodactyl"]}>
        <Tab>
            右鍵 `run.ps1`，並點擊 `Run with Powershell`

            ![啟動 run.ps1](@static/blog/2023-make-music-bot-without-coding/run-powershell.png)
        </Tab>
        <Tab>
            運行 `run.sh`

            ```bash
            cd LavaLauncher
            ./run.sh
            ```
            <Tip> 如果你遇到了 Permission Error，試著 `sudo chmod +x run.sh` </Tip>
        </Tab>
        <Tab>
            直接運行 `run.py`

            ```bash
            python3 run.py
            ```
        </Tab>
    </Tabs>

    3. 輸入資料

    [LavaLauncher](https://github.com/Nat1anWasTaken/LavaLauncher) 的第一次啟動可能需要一點時間，請耐心等待
    在啟動完成後，你會看到幾個彈出來的問題

    - `[?] Please enter the port for lavalink:` <br/>
    請輸入你想要的 Lavalink 連接埠，預設即可，Pterodactyl 面板請輸入任何一個開放的 Port

    - `[?] TOKEN:`
    請輸入你在上方 [創建機器人](#創建-discord-機器人) 獲得的 TOKEN

    - `[?] Do you want to enable Spotify support?`
    請輸入 `y` 或 `n` 來決定是否啟用 Spotify 支援

    - `[?] SPOTIFY_CLIENT_ID:`, `[?] SPOTIFY_CLIENT_SECRET:`
    請輸入你在上方 [設置 Spotify](#設置-spotify) 獲得的 Client ID 和 Client Secrets

    `[?] SPOTIFY_REDIRECT_URI:`
    請輸入 `http://localhost/`

    接著，LavaLauncher 會跳出一個連結，請點擊它並授權
    授權完成後，顯示 `http://localhost/ 拒絕連線` 是正常的，此時你應該將瀏覽器分頁的連結複製下來，並貼到 LavaLauncher 的輸入框中

    4. 運行機器人

    [LavaLauncher](https://github.com/Nat1anWasTaken/LavaLauncher) 會自動安裝所需的套件，並啟動機器人
    如果你看到 `Logged in as` 這行字，恭喜你，你已經成功啟動了 Lava

</div>
