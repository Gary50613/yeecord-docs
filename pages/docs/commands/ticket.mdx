# 私人頻道

import Adsense from "@components/mdx/Adsense";

當你正在舉辦一個抽獎，或是希望各位能回報問題給你，但是又不希望其他人私訊你的時候，私人頻道能夠解決這些困擾，在伺服器上創建屬於一個人和管理員的溝通頻道

## 初始化私人頻道

首先請確認機器人擁有 `管理頻道` 的權限，才能避免發生權限不足的錯誤！

使用指令 `&ticket` 之後會跳出支票系統的選單，點擊 `初始化支票系統` 之後，會出現一個父頻道叫 `支票系統` 以及一個文字頻道 `創建支票`，你可以自己改成你喜歡的名字。

<Adsense />

## 創建

點擊 `創建支票` 頻道內的 `創建私人頻道` 按鈕，將會生成一個頻道，該頻道只有點選按鈕的成員以及 **擁有管理員權限的人** 才能看到以及回覆，一般人並不會有權限查看。

## 刪除

點擊私人頻道裡的第一則訊息下方 `刪除支票` 的按鈕，並且再次點選 `我確定` 刪除。
