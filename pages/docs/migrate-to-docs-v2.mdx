### 我的頁面在哪裡??

所有 mdx 檔案都已移至 `/pages`

| 文檔          | 部落格        |
| ------------- | ------------- |
| `/pages/docs` | `/pages/blog` |

### 圖片引入方法更新

以前的版本允許您從 `/static` 資料夾引入圖片, 例如:

```mdx
![說明文字](/my-photo.png)
```

現在你必須將你的圖片放入 `/public` 資料夾並使用 `![alt](@static/my-photo.png)` 導入。

```mdx
# 以前

![alt](/docs/my-photo.png)

# 現在

![alt](@static/docs/my-photo.png)
```

在 `/pages/blog` 資料夾中建立檔案

```mdx
---
title: 你的標題

authors: [<作者編號>]
```

#### 得到你自己的作者編號

打開`config.ts`文件, 滾動到`blogAuthors`並添加您的個人信息

-   避免文字太長
-   不要放可疑的網址

```ts
export const blogAuthors: Record<string, AuthorData> = {
    //...
    henry: {
        name: "你的名字",
        title: "簡短的自我介紹",
        url: "https://my-website.com",
        image_url: "/blog/henry.jpg", //將圖像放入 /public/blog/henry.jpg
    },
};
```

之後你便可以用`henry`作為你的作者編號

### 創建內容

我相信你已經學習了一些基本的 Markdown 語法

#### 你需要注意這些事情

-   適當的用詞
-   禁止自我宣傳
-   不應帶偏見或政治立場

#### 改進你的文章

-   **漂亮的排版:**
    使用標題而不是大量雜亂無章的文字
-   **數字和英文之間的空格:**
    一切都遵循 [中文案排版指南](https://github.com/sparanoid/chinese-copywriting-guidelines)
-   **使用自定義組件:**
    您可以從 [這裡](#更多自定義組件) 了解更多信息

### 排序頁面

您必須編輯 `_meta.json` 以更改每個頁面的順序：

```json
{
    "my-new-page": "新頁面標題",
    "index": "快速入門",
    "commands": "指令"
}
```

## 更多自定義組件

使用 `import { ... } from "@components/mdx"` 輕鬆導入自定義組件

### Accordion

想告訴使用者額外的詳細資訊？使用 `Accordion` 組件！

```mdx
import { Accordion } from "@components/mdx";

**這裡有一條鯊魚**

<Accordion title="Money是一隻鯊魚">Yeahhhhh</Accordion>;
```

**這裡有一條鯊魚**

<Accordion title="Money是一隻鯊魚">Yeahhhhh</Accordion>

### Admonition

想給使用者提示或警告? 你可以使用 `Admonition` 組件

它支持多種類型：`Tip`、`Warning`、`Info`、`Error`：

```mdx
import { Admonition, Error, Tip, Warning, Info } from "@components/mdx";

<Admonition title="Default" />
<Tip title="Tip" />
<Warning title="warning" />
<Info title="info" />
```

<Admonition title="Default" />
<Tip title="Tip" />
<Warning title="warning" />
<Info title="info" />

### Card

您可以使用它來將使用者重定向到不同的頁面，記得傳遞 `href` 道具

```mdx
import { Card } from "@components/mdx";

<Card title="重定向到部落格" href="/blog/welcome-to-v2" />
<Card title="重定向到部落格" href="/blog/welcome-to-v2" arrow />
```

## 建立自己的組件

Mdx 能夠與 React 和 Next.js 一起使用，您可以在 `.mdx` 檔案中使用 jsx 語法。

同時我們將 Tailwind CSS 用於內部的 css 樣式，這意味著你可以輕鬆地建立組件。

<Tip title="重要提示">請確保它們同時支持淺色和深色模式</Tip>
### 我們正在使用這些函式庫

你可以在你的文檔中使用它們!

-   [framer-motion](https://www.framer.com/motion/)
-   `nextra`
-   `nextra-docs-theme`
-   `react-youtube`

### Framer Motion

它可以讓您輕鬆建立炫酷的動畫

```jsx
/>
```
