| 文檔          | 部落格          |
| ------------- | ------------- |
| `/pages/docs` | `/pages/blog` |

### 圖片引入方法更新

以前的版本允許您從 `/static` 資料夾引入圖片, 例如: `![說明文字](/my-photo.png)`

現在你必須將你的圖片放入 `/public` 資料夾並使用 `![alt](@static/my-photo.png)` 導入它。
同時你也可以把圖片放到你的 mdx 檔案的同一個資料夾中, 並使用 `![alt](./my-photo.png)`

```mdx
# 以前

![alt](/docs/my-photo.png)

# 現在

![alt](@static/docs/my-photo.png)
```

## 更多自定義組件

使用 `import { ... } from "@components/mdx"` 輕鬆導入自定義組件

### Accordion

想告訴使用者額外的詳細資訊? 使用 `Accordion` 組件!

```mdx
import { Accordion } from "@components/mdx";

**這裡有一條鯊魚**

<Accordion title="Money是一隻鯊魚">Yeahhhhh</Accordion>;
```

**這裡有一條鯊魚**

<Accordion title="Money是一隻鯊魚">Yeahhhhh</Accordion>

### Admonition

想給使用者提示或警告? 你可以使用 `Admonition` 組件，它還支持多種類型：`Tip`、`Warning`、`Info`、`Error`：

```mdx
import { Admonition, Error, Tip, Warning, Info } from "@components/mdx";

<Admonition title="Default" />
<Tip title="Tip" />
<Warning title="warning" />
<Info title="info" />
```

<Admonition title="Default" />
<Tip title="Tip" />
<Warning title="warning" />
<Info title="info" />

### Card

您可以使用它來將使用者重定向到不同的頁面 <br />
記得傳遞 `href` 道具

```mdx
import { Card } from "@components/mdx";

<Card title="重定向到部落格" href="/blog/welcome-to-v2" />
<Card title="重定向到部落格" href="/blog/welcome-to-v2" arrow />
```

## 建立自己的組件

Mdx 能夠與 React 和 Next.js 一起使用, 您可以在 `.mdx` 檔案中使用 jsx 語法 <br />
同時我們將 Tailwind CSS 用於內部的 css 樣式, 他們支持黑暗模式 <br />
所以意味著你可以輕鬆地建立組件

我們也正在使用這些庫： [framer-motion](https://www.framer.com/motion/), `nextra`, `nextra-docs-theme`, `react-youtube` <br />
你可以在你的文檔中使用它們

### 自定義組件 (Tailwind CSS)

```jsx
<div className="p-3 border-black dark:border-white border-2">Hello World</div>
```

<div className="p-3 border-black dark:border-white border-2">Hello World</div>

### Framer Motion

它可以讓您輕鬆建立炫酷的動畫

```jsx
    animate={{
        scale: [1, 2, 2, 1, 1],
        rotate: [0, 0, 270, 270, 0],
        borderRadius: ["20%", "20%", "50%", "50%", "20%"],
    }}
/>
```
