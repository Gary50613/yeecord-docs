## 新增頁面的方法

我們使用 Fumadocs 來製作我們的文檔，您可以參考 [他們的指南](https://fumadocs.vercel.app/docs/ui/markdown) 來了解它的更多功能。

### 建立 MDX 檔案

請填寫所有字段

<Tabs items={["文檔", "部落格"]}>

<Tab value="文檔">

在 `/content/docs` 資料夾中建立檔案

```mdx title="page.mdx"
---
title: 你的標題
description: 這是關於什麼的?
---

內文...
```

</Tab>

<Tab value="部落格">
  
在 `/content/blog` 資料夾中建立檔案

```mdx title="page.mdx"
---
title: 你的標題
description: 這是關於什麼的?
authors: [<作者編號>]
date: 2023/1/1
image: "/blog/banner.jpg"
tags: ["文章標籤"]
---

內文...
```

<h4>得到你自己的作者編號</h4>

打開 `config.ts` 文件, 滾動到 `blogAuthors` 並添加您的個人信息

- 避免文字太長
- 不要放可疑的網址

```ts
export const blogAuthors: Record<string, AuthorData> = {
  //...
  henry: {
    name: "你的名字",
    title: "簡短的自我介紹",
    url: "https://my-website.com",
    image_url: "/blog/henry.jpg", //將圖像放入 /public/blog/henry.jpg
  },
};
```

之後你便可以用 `henry` 作為你的作者編號

### 創建內容

我相信你已經學習了一些基本的 Markdown 語法

#### 你需要注意這些事情

- 適當的用詞
- 禁止自我宣傳
- 不應帶偏見或政治立場

#### 改進你的文章

- **漂亮的排版:**
  使用標題而不是大量雜亂無章的文字
- **數字和英文之間的空格:**
  一切都遵循 [中文案排版指南](https://github.com/sparanoid/chinese-copywriting-guidelines)

### 排序頁面

您必須編輯 `meta.json` 以更改每個頁面的順序：

```json
{
  "pages": ["index", "my-new-page"]
}
```
